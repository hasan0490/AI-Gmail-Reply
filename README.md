# AI Gmail Reply Assistant

An AI-powered Gmail automation built with n8n and Groq AI.

## Features

- Automatically detects new Gmail messages
- Generates professional AI-powered replies
- Creates Gmail drafts instead of sending emails automatically
- Allows manual review before sending
- Built using a no-code/low-code workflow

## Workflow

New Gmail
      ↓
AI Agent (Groq)
      ↓
Generate Reply
      ↓
Create Gmail Draft

## Tech Stack

- n8n
- Gmail API
- Groq API

## Installation

1. Download the workflow JSON.
2. Import it into n8n.
3. Connect Gmail credentials.
4. Add your Groq API key.
5. Activate the workflow.
